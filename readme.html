User-Flow

  • Create activity
  • Set name, multiplier and minimal time
  • (Optional) Add activity to category (otherwise in root)
  • (Optional) Add category to category
  • Click "give activity"
  • (Optional) Choose category to choose from (otherwise defaults to root)
  • (Optional) Insert minimal time (otherwise defaults to 0 seconds)
  • After receiving activity, press "start" to start counting time
  • Press "stop" to stop timer
  • (Optional: Change counted time)
  • Press "add time to activity" (or cancel)
  • (Optional) Insert custom time to chosen activity (to use as tracker instead of recommender)
